# Contributing to EcoSort AI

We love your input! We want to make contributing to this project as easy and transparent as possible.

## Pull Requests

1. Fork the repo and create your branch from `main`.
2. If you've added code that should be tested, add tests.
3. Ensure the test suite passes.
4. Issue that pull request!

## Regional Rules Contributions

To add a new city or region's recycling rules, please see our rules engine schema in `client/public/rules/global.json` and submit a PR with your new file (e.g. `rules/japan.json`).
